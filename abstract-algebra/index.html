抽象代数笔记

Posted by WHZ0325 on 2022-11-09, Viewed times

按我们老师讲的内容来，可能不太全……

一、代数系统

  1. 代数运算：封闭性。
  2. 代数运算的性质：
    1. 交换律
    2. 结合律
    3. 幂等律：集合中每个元素都是幂等元（满足 $a*a=a$ 的 $a$）。【拓展】满足 $a^n=a$。
    4. 分配律：若 $a*(b+c)=ab+ac$ 表明“乘法”对“加法”满足分配率（左分配）。
    5. 吸收律：若 $a*(a+b)=a+(ab)=a$ 表明二元运算 $$ 与 $+$ 满足吸收率，$P\and (P\or Q)=P\or (P\and Q)=P$，$\min{a,\max{a,b}}=\max{a,\min{a,b}}=a$。
    6. 消去律：若 $ac=bc$ 则 $a=b$ 表明二元运算 $*$ 满足消去律（区分：左消去/右消去）。
  3. 代数系统：$(S,f_1,f_2,\dots)$，其中 $S$ 为非空集合，$f_1,f_2,\dots$ 为代数运算。

二、群的定义

  1. 半群 $(G,·)$：
    1. 集合非空
    2. 代数运算（封闭）
    3. 满足结合律，如 $(G,·)$。
  2. 群 $(G,·)$：
    1. 半群
    2. 有单位元
    3. 任意元素有逆元素
  3. 理解群的定义：
    1. 单位元是群中唯一幂等元。
    2. 群中不可能有零元（半群可以）。【证明】反证法：群中只有一个元素时该元素被视为单位元，否则矛盾。
    3. 群一定满足消去律。【证明】两边同时乘被消去元素的逆元（左消去左乘，右消去右乘）。
  4. 群的性质：
    1. 群中单位元唯一，每个元素的逆元素唯一。
    2. 群的证明可以简化为：有左壹，有左逆的半群（也可以简化为：有右壹，有右逆）（不可以简化为：一个左，一个右）。
    3. 群的证明可以等价为满足可除条件的半群：对任意 $a,b$，存在 $x,y$ 满足 $x·a=b$ 且 $a·y=b$。
    4. 群的运算可以任意加括号。
    5. $a^n$：规定 $a^0=1$，$a^{-n}=(a^n)^{-1}$。满足第一指数律（$a^m·a^n=a^{m+n}$）和第二指数律（$(a^m)^n=a^{mn}$），不一定满足第三指数律（$(a·b)^n=a^n·b^n$），因为不一定满足交换律。
  5. Abel 群：满足交换律的群，也称为交换群。
    1. 交换群的运算可以任意改变次序。
    2. 交换群满足第三指数律。
    3. 另：我们永远假定加法群是交换群。
